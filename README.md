# 🚗 Tawseel - منصة توصيل ذكية

## الميزات
- خرائط تفاعلية (OpenStreetMap)
- نظام أدوار (زبون، سائق، مشرف)
- دفع نقدي ونجوم تيليجرام
- تتبع مباشر
- لوحة تحكم إدارية

## التقنيات
- React + TypeScript + Vite
- Supabase + PostGIS
- Vercel Serverless
- Telegram Mini App

## النشر
1. انسخ `.env.example` إلى `.env` واملأ المتغيرات.
2. `cd backend && npm install && vercel --prod`
3. `cd frontend && npm install && npm run build && npm run deploy`
